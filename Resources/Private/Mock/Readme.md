# Mock Server for Event Submission API

## Requirements
* docker (compose)
* curl, HTTPie or any other tool for HTTP requests

## Usage

### Start Mock Server

```shell
cd Resources/Private/Mock
docker compose up
```
Note: Run `docker compose up -d` in order to detach from the output of the container.
